Bootstrap Framework

The world's most popular front-end component library for building responsive, mobile-first websites.

What is Bootstrap?

Bootstrap is a powerful, feature-packed frontend toolkit. Build anything from prototype to production in minutes.

Responsive Design

Bootstrap's responsive CSS adjusts to phones, tablets, and desktops with a single code base.

Rapid Development

Quickly prototype your ideas or build your entire app with our Sass variables and mixins.

Component Library

Tons of components that are responsive and customizable for any project.

Bootstrap Code Examples

Basic Bootstrap Setup

<!-- Include Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap container example -->
<div class="container">
  <div class="row">
    <div class="col-md-6">
      Column 1
    </div>
    <div class="col-md-6">
      Column 2
    </div>
  </div>
</div>

<!-- Include Bootstrap JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Evolution of Bootstrap

2011

Bootstrap 1

Originally created by Twitter developers as an internal tool, later released as an open-source project.

2013

Bootstrap 3

Mobile-first approach, flat design, and rewritten components.

2018

Bootstrap 4

Major rewrite with Flexbox, new components, and ES6 JavaScript.

2021

Bootstrap 5

Dropped jQuery, new utilities, improved customization, and vanilla JavaScript.

Bootstrap Best Practices

Do's

  • Use the grid system for responsive layouts
  • Customize with Sass variables
  • Utilize utility classes for rapid development
  • Keep components accessible
  • Use responsive breakpoints appropriately

Don'ts

  • Don't override Bootstrap styles unnecessarily
  • Avoid nesting containers
  • Don't forget to include Popper.js for components
  • Avoid using !important in your CSS
  • Don't ignore mobile-first approach
